# Advanced Calculator

A menu-driven Java application providing a variety of mathematical utilities and numerical analysis functions. The project was developed to practice problem solving and modular programming.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Even/Odd Detection
- Prime Number Checking
- Fibonacci Series Generation
- Sum of First N Numbers
- Multiplication Tables
- Minimum and Maximum Finder
- Average Calculation

## Technologies Used

- Java
- Functions and Methods
- Loops
- Conditional Statements
- User Input Handling

## Learning Outcomes

This project was developed to strengthen understanding of:

- Java Fundamentals
- Problem Solving
- Modular Programming
- Mathematical Algorithms
- User Interaction Design
